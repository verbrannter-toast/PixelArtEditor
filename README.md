# PixelArtEditor

This project got developed as a C++ uni project by Erik Mettner. It is a simple Pixel Art Editor that features a color picker, different brush shapes and sizes and an option to resize the canvas up to 999 * 999 pixels. 

## How to run

The project requires SFML (Version 3.0.2) so make sure to have that installed. Apart from that, a custom font is used that should be recognized by default.

If the font isn't automatically recognized, make sure to navigate to `Run > Edit Configurations > Working Directory` and set it to the project root.

## Controls

- LMB to draw pixels
- RMB to erase pixels
- MMB and dragging to pan across the canvas
- Scrolling to zoom in and out

Apart from these, there are multiple buttons and panels that can be used for different functionalities but those are pretty self-explanatory or have tips directly written near them.
